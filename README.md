# VerifyPolitics

Prototype website for a browser extension that helps users evaluate the reliability of political information online.

## Project goal
Create a modern landing page to present the concept of the extension, explain how it works, and support prototype testing with users.

## Main problem
Young users are exposed to political content online but often lack time, clarity, and trust to verify if the information is reliable.

## Proposed solution
A browser extension that:
- detects political content
- displays a reliability score
- provides warning signals
- gives access to sources and explanations

## Current project structure
- `index.html` → landing page
- `css/style.css` → styles
- `js/script.js` → interactions
- `assets/` → images, icons, mockups
- `docs/` → project documentation and copywriting
